# SortingGame_React
สามารถลองเล่นได้ที่ http://82.26.104.109:3000/ ได้ถึงวันที่ 5/10/25

### ติดตั้ง dependencies
```
cd backend && npm install
cd ../frontend && npm install
```
### รัน Frontend (Next.js)
 ```
cd frontend
npm run dev
```
### รัน Backend (NestJS)
```
cd backend
npm run start:dev
```
